QtitanRibbon quick start guide

We did our best to make the process of installing and using QtitanRibbon for Qt/C++ as easy and hassle-free as at all possible. For creation of this tutorial we use: OS Windows 8 64 bit, Qt 5.2.0 (for Windows 32 bit), QtCreator 3.0.0 (for Mingw) and QtitanRibbon 3.2 (for Mingw). Below is a description of several simple steps you need to take to start using of QtitanRibbon in your own Qt/C++ application:

  • Download and install QtitanRibbon for QtCreator (mingw) from this page - http://www.devmachines.com/qtitanribbon-overview.html . In our case install folder is "c:\projects\QtitanRibbon3.2.0_mingw"
  • Start QtCreator and open <QtitanRibbon Install Folder>/qtitanribbon.pro via main menu. QtCreator will allow to configure the project. See screenshot 1. Press "Configure Project" button.

  • After configuring you will see some errors (such as "cannot read /src/shared/qtitanribbon.pri: No such file or directory"). It is normal and not a problem on this step. See screenshot 2.

  • Switch to the Release mode. QtitanRibbon demo build does not contain any debug libraries so it is necessarily to continue. If you have a purchased license this step is entirely optional. See screenshot 3.

  • Open "Projects" tab at the left of QtCreator view and scroll down to the Build Environment. Click "Details" button and add QTITANDIR variable to the environment with <QtitanRibbon Install Folder> value. In our case it is QTITANDIR=c:\projects\QtitanRibbon3.2.0_mingw. See screenshot 4.

  • Run "qmake" command from the main menu. See screenshot 5.

  • If all is Ok, qmake will be completed without any errors. See screenshot 6.

  • Run "Build All" command from the main menu. See screenshot 7.

  • If all is Ok, the build will be completed and you can start QtitanRibbon demo expample. See screenshot 8.

  • Press Ctrl+R to start QtitanRibbon demo expample. See screenshot 9.

We have considered an example of how to configure, compile and run a standart demo example from QtitanRibbon installation. Next, consider an example of creating an application containing the Ribbon Bar from scratch.

  • Create Qt Widgets Application with name "qtitanribbon_application" from the QtCreator's main menu. See screenshot 10.

  • Press "Next" to follow next step of creation wizard. See screenshot 11.

  • Press "Next". See screenshot 12.

  • Press "Finish". See screenshot 13.

  • Empty Qt application has been created. See screenshot 14.

  • Switch to the Release mode. QtitanRibbon demo build does not contain any debug libraries so it is necessarily to continue. If you have a purchased license this step is entirely optional. See screenshot 15.

  • Open "qtitanribbon_application.pro" file and add include($$(QTITANDIR)/src/shared/qtitanribbon.pri) to it.
        QT       += core gui

    greaterThan(QT_MAJOR_VERSION, 4): QT += widgets

    TARGET = qtitanribbon_application
    TEMPLATE = app


    SOURCES += main.cpp\
            mainwindow.cpp

    HEADERS  += mainwindow.h

    FORMS    += mainwindow.ui

    include($$(QTITANDIR)/src/shared/qtitanribbon.pri)

    See screenshot 16.

  • Open "Projects" tab at the left of QtCreator view and scroll down to the Build Environment. Click "Details" button and add QTITANDIR variable to the environment with <QtitanRibbon Install Folder> value. In our case it is QTITANDIR=c:\projects\QtitanRibbon3.2.0_mingw. See screenshot 17.

  • Run "qmake" command from the main menu. See screenshot 18.

  • If all is Ok, qmake will be completed without any errors. See screenshot 19.

  • Open "main.cpp" and add some code to add RibbonStyle to the application.
        #include "mainwindow.h"
    #include <QApplication>
    #include <QtitanRibbon.h>

    int main(int argc, char *argv[])
    {
        QApplication a(argc, argv);
        a.setStyle(new Qtitan::RibbonStyle());
        MainWindow w;
        w.show();

        return a.exec();
    }

    See screenshot 20.

  • Rebuild project "qtitanribbon_application" from the main menu. See screenshot 21.

  • If all is Ok, the build will be completed without any issue you can start "qtitanribbon_application". See screenshot 22.

  • Press Ctrl+R to start "qtitanribbon_application". See screenshot 23.

We hope these examples will help you to quickly start using the component in your application. In any case, if you have any questions or problems please let us know by this mailbox - support@devmachines.com.
